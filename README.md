## nand2tetris

Projects for Nand2Teris.

Nand2Tetris website: https://www.nand2tetris.org/

---

### Project06 Assembler

#### Compilation
```
$ g++ -std=c++17 assembler.cpp -o assembler
```

#### Usage
```
$ assembler filename.asm
```

It generates filename.hack
